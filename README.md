# akkomadeck

a tweetdeck-like client for akkoma(but, has some compatible for Mastodon) 

demo: [from vercel.app](https://akkomadeck.vercel.app/)

## features

- [x] boost
- [x] favorites
- [ ] show CW
- [ ] post
- [ ] emoji reaction
- [ ] add column
  - list TL
  - federated TL
  - local TL
- [ ] customize column order
